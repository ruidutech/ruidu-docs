## 接口详情

### 请求声音列表

**简要描述:**
获取所有已上传声音的列表

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/audio_manage/get_audio_list`

**请求方式:**
- GET 
- POST

**请求参数:**
无

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": {
    "audios": [
      "audio1.wav",
      "audio2.wav"
    ]
  },
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| audios | array | 音频文件名称列表 |

**备注:**
更多返回错误代码请看首页的错误代码描述

**请求URL:** 
- `http://机器人IP:端口号/robot/audio_manage/play_audio?audio_name=`

- GET 
- POST

**请求参数:**
| 参数名 | 是否必须 | 类型 | 说明 |
|:-------|:--------:|:----:|:-----|
| audio_name | 是 | string | 音频文件名 |

**返回参数说明:**

| errorCode | int | 错误代码 |

### 上传音频文件

**简要描述:**
上传音频文件到机器人

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/audio_manage/upload_audio`

**请求方式:**
- POST

**请求参数:**
| 参数名 | 是否必须 | 类型 | 说明 |
|:-------|:--------:|:----:|:-----|
| file | 是 | file | 音频文件（.wav格式） |

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": {
    "audio_name": "audio1.wav"
  },
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "audio already exists",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| errorCode | int | 错误代码 |
| audio_name | string | 上传后的音频文件名 |
