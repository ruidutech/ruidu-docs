### 添加停车位

**简要描述:**
添加新的停车位信息

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/parking/add_parking`

**请求方式:**
- GET 
- POST

**请求参数:**
| 参数名 | 是否必须 | 类型 | 说明 |
|:-------|:--------:|:----:|:-----|
| parking_info | 是 | string | 停车位信息，JSON字符串 |

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": {
    "parking_id": "12345"
  },
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| parking_id | string | 停车位ID |

**备注:**
更多返回错误代码请看首页的错误代码描述

### 获取停车位

**简要描述:**
获取停车位信息

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/parking/get_parking?parking_id=`

**请求方式:**
- GET 
- POST

**请求参数:**
| 参数名 | 是否必须 | 类型 | 说明 |
|:-------|:--------:|:----:|:-----|
| parking_id | 否 | string | 停车位ID，不传时返回所有停车位 |

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": {
    "parkings": [
      {
        "parking_id": "12345",
        "name": "停车位1",
        "position": {
          "x": 1.23,
          "y": 4.56
        },
        "orientation": {
          "x": 0.0,
          "y": 0.0,
          "z": 0.7071,
          "w": 0.7071
        },
        "status": 0
      }
    ]
  },
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| parkings | array | 停车位列表 |
| parking_id | string | 停车位ID |
| name | string | 停车位名称 |
| position | object | 位置信息 |
| orientation | object | 方向信息（四元数） |
| status | int | 状态：0-空闲，1-占用 |

**备注:**
更多返回错误代码请看首页的错误代码描述
