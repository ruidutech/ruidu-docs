### 开始录制bag

**简要描述:**
开始录制数据包

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/bag_manage/record_bag?bag_name=&&type=`

**请求方式:**
- GET 
- POST

**请求参数:**
| 参数名 | 是否必须 | 类型 | 说明 |
|:-------|:--------:|:----:|:-----|
| bag_name | 是 | string | 数据包名称 |
| type | 是 | string | 录制类型，可选值：nav（导航模式）、slam（建图/定位） |

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": "",
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| errorCode | int | 错误代码 |

**备注:**
更多返回错误代码请看首页的错误代码描述

### 获取bag列表

**简要描述:**
获取已录制的数据包列表

**接口版本:**
| 版本号 | 制定日期 | 修订日期 |
|:-------|:--------:|:--------:|
| 1.0.0  | 2020-9-11 | 2020-9-11 |

**请求URL:** 
- `http://机器人IP:端口号/robot/bag_manage/get_bag_list`

**请求方式:**
- GET 
- POST

**请求参数:**
无

**返回示例:**

正确时返回:
```json
{
  "errorCode": "",
  "msg": "successed",
  "data": {
    "bags": [
      {
        "name": "bag1.bag",
        "size": 1024000,
        "time": "2022-09-10 10:20:30"
      },
      {
        "name": "bag2.bag",
        "size": 2048000,
        "time": "2022-09-11 11:21:31"
      }
    ]
  },
  "successed": true
}
```

错误时返回:
```json
{
  "errorCode": 100,
  "msg": "",
  "data": "",
  "successed": false
}
```

**返回参数说明:**
| 参数名 | 类型 | 说明 |
|:-------|:-----|:-----|
| bags | array | 数据包列表 |
| name | string | 数据包名称 |
| size | int | 数据包大小，单位字节 |
| time | string | 创建时间 |
