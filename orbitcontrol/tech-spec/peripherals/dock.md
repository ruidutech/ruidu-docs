# 机巢 API

## 通信协议

- **通信接口**: RS-485
- **协议类型**: Modbus RTU
- **校验方式**: CRC16
- **波特率**: 9600
- **电机控制器地址**: 0x07

## 开舱控制

控制机巢舱门打开。

- **功能码**: 0x06（写单个寄存器）
- **寄存器地址**: 0x0000
- **命令**: 0x0001（开）
- **完整报文**: `07 06 00 00 00 01 48 6C`

| 字节 | 值     | 说明         |
| :--- | :----- | :----------- |
| 1    | 0x07   | 从站地址     |
| 2    | 0x06   | 功能码       |
| 3-4  | 0x0000 | 寄存器地址   |
| 5-6  | 0x0001 | 写入值（开） |
| 7-8  | 0x486C | CRC16 校验   |

## 关舱控制

控制机巢舱门关闭。

- **功能码**: 0x06（写单个寄存器）
- **寄存器地址**: 0x0000
- **命令**: 0x0000（关）
- **完整报文**: `07 06 00 00 00 00 89 AC`

| 字节 | 值     | 说明         |
| :--- | :----- | :----------- |
| 1    | 0x07   | 从站地址     |
| 2    | 0x06   | 功能码       |
| 3-4  | 0x0000 | 寄存器地址   |
| 5-6  | 0x0000 | 写入值（关） |
| 7-8  | 0x89AC | CRC16 校验   |

## 响应格式

控制器正常响应时返回：

- **开舱响应**: `07 06 00 00 00 01 48 6C`
- **关舱响应**: `07 06 00 00 00 00 89 AC`

响应报文与命令报文相同（Modbus 标准响应）。
